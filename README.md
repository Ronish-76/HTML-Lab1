# HTML Lab 1 - Student Information Website

## Assignment Title

HTML Lab 1: Creating a Student Information Webpage

## Brief Description

This project contains a series of HTML exercises demonstrating fundamental web development concepts. The main deliverable is a student information webpage (Exercise 4) that showcases various HTML elements including:

- Semantic HTML structure (header, nav, article, footer)
- Navigation links
- Text formatting with different font families
- Student information display
- Proper HTML document structure

The project includes 4 progressive exercises that build upon HTML fundamentals, with supporting assets including images and audio files.

## Project Structure

```
Exercise1.html       - First HTML exercise
Exercise2.html       - Second HTML exercise
Exercise3.html       - Third HTML exercise
Exercise4.html       - Main student information webpage
assets/
  ├── images/        - Image files used in exercises
  ├── audios/        - Audio files used in exercises
  └── css/           - CSS stylesheets (if applicable)
```

## Steps to Run the Project

1. **Clone the Repository**

   ```bash
   git clone https://github.com/Ronish-76/HTML-Lab1.git
   cd HTML-Lab1
   ```

2. **Open the Project**

   - Navigate to the project folder in your file explorer
   - Double-click on any `.html` file to open it in your default browser
   - Or right-click and select "Open with" to choose a specific browser

3. **View the Main Exercise**

   - Open `Exercise4.html` to see the student information webpage
   - This is the main deliverable showing a complete HTML page structure

4. **Navigate Between Exercises**
   - Use the navigation menu in Exercise 4 to move between different exercises
   - Click on "Home", "About Us", or "Contact" links

## Browser Compatibility

- Chrome/Edge (recommended)
- Firefox
- Safari
- Any modern HTML5-compatible browser
